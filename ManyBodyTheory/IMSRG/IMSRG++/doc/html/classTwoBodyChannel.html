Detailed Description

Two-body channel is a collection of Kets having same J,P,T in the model space

channel has Kets { |pq> } which can have same J,P,T. p and q can be particle/hole and core/valence/q

construct two-body channels from model space

J, parity, Tz are infered from model space as

J = N%(tbjmax+1);

parity = (N/(tbjmax+1))%2;

Tz = (N/(2*(tbjmax+1))-1);

collect kets from model space for two-body channel { |pq>_{JPT} }

  1. J = N%(tbjmax+1) # it is not clear how N is defined..
  2. parity = (N/(tbjmax+1))%2;
  3. Tz = (N/(2*(tbjmax+1))-1)
  4. make KetMap from model space Kets by CheckChannel_ket(ket)
  5. Get KetIndex_pp in this channel from modelspace KetIndex_pp. other index of _hh,_ph,_cc ..in a similar way.
  6. Get Ket_occ_hh and Ket_unocc_hh in this channel from model space. Ket_occ_ph Ket_unocc_ph in a similar way
