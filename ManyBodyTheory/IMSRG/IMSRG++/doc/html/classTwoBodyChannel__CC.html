check if |pq> participates in this channel

Check if orbits pq participate in this cross-coupled two-body channel

Difference from regular channels: no Pauli rule, <pp||nn> is allowed. But |Tz| is still conserved, i.e. <pp||pn> is not allowed. So we use |Tz| rather than Tz, and don't use Tz=-1.
